## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

1. start data base
```bash
$ docker-compose up -d
```
2. Install package node
```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
